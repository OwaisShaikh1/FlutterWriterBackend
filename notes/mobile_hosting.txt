# 🧱 FULL SETUP GUIDE — PHONE AS BACKEND SERVER

---

# 🔹 PHASE 1 — One-Time Setup (Only Do Once)

---

## 1️⃣ Install Termux (F-Droid version recommended)

Then inside Termux:

```bash
pkg update && pkg upgrade
pkg install nodejs mariadb tmux
```

---

## 2️⃣ Initialize MariaDB (Only First Time)

```bash
mariadb-install-db --user=$(whoami)
```

If it says already initialized → good.

---

## 3️⃣ Secure MariaDB (Optional but Recommended)

```bash
mysql_secure_installation
```

If you don’t set root password, leave DB_PASSWORD empty in `.env`.

---

## 4️⃣ Create Your Database

```bash
mysql -u root
```

Inside:

```sql
CREATE DATABASE writer CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
EXIT;
```

---

## 5️⃣ Import Dump (If Migrating From Laptop)

Copy dump via USB to:

```
Internal Storage → Download
```

Enable storage:

```bash
termux-setup-storage
```

Import:

```bash
mysql -u root writer < ~/storage/downloads/your_dump.sql
```

---

# 🔹 PHASE 2 — Backend Configuration

---

## 6️⃣ `.env` File (Inside Backend Folder)

Create `.env`:

```env
DB_HOST=127.0.0.1
DB_USER=root
DB_PASSWORD=
DB_NAME=writer
JWT_SECRET=your_secret_here
PORT=5000
```

---

## 7️⃣ Express Must Listen on All Interfaces

At bottom of `index1.js`:

```js
const PORT = process.env.PORT || 5000;
app.listen(PORT, '0.0.0.0', () =>
  console.log(`Server running on port ${PORT}`)
);
```

---

# 🔹 PHASE 3 — Network Setup (Hotspot Mode)

---

## 8️⃣ Turn ON Hotspot

Phone Settings → Hotspot ON

---

## 9️⃣ Find Phone IP

In Termux:

```bash
ip addr
```

Look for:

```
ap0
inet 192.168.xx.xx
```

Example:

```
192.168.66.19
```

That is your backend IP.

---

# 🔹 PHASE 4 — Daily Startup Script (IMPORTANT)

Every time phone restarts:

---

## 🔟 Start tmux Session

```bash
tmux
```

---

## 1️⃣1️⃣ Start MariaDB

```bash
mysqld_safe &
```

Wait 5 seconds.

Test:

```bash
mysql -u root
exit;
```

---

## 1️⃣2️⃣ Start Backend

```bash
node index1.js
```

You should see:

```
Connected to MySQL!
Server running on port 5000
```

---

## 1️⃣3️⃣ Detach tmux

```
Ctrl + B → D
```

Server keeps running in background.

To return later:

```bash
tmux attach
```

---

# 🔹 PHASE 5 — Flutter Configuration

---

## 1️⃣4️⃣ Set Base URL

In Flutter:

```dart
final baseUrl = "http://192.168.66.19:5000";
```

Replace with current IP.

⚠ If IP changes when hotspot restarts → update it.

# 🔹 PHASE 7 — Stability Settings (CRITICAL)

---

## 1️⃣6️⃣ Disable Battery Optimization

Settings → Apps → Termux → Battery →
Set to **Unrestricted**

---

## 1️⃣7️⃣ Backup Strategy

Run periodically:

```bash
mysqldump -u root writer > backup.sql
```

Copy backup to laptop.

Phones are not reliable storage.

# 🔹 OPTIONAL IMPROVEMENTS (Recommended)

---

## Use Connection Pool Instead of Single Connection

Replace:

```js
mysql.createConnection
```

With:

```js
mysql.createPool({
  host: process.env.DB_HOST,
  user: process.env.DB_USER,
  password: process.env.DB_PASSWORD,
  database: process.env.DB_NAME,
  waitForConnections: true,
  connectionLimit: 10,
});
```

More stable.

---

## Add Health Endpoint

```js
app.get('/health', (req, res) => {
  res.json({ status: 'ok' });
});
```

Test with:

```
http://IP:5000/health
```
